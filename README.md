# Self-Improvement App

## Features
- **Daily Reflection**: Answer 3 questions every night.
- **Meditation**: Timer for meditation sessions.
- **Reading**: Timer and notes for reading sessions.
- **Planning**: Plan 3 important tasks for the next day.
- **Dashboard**: Visual insights into your progress with 9 interactive charts.

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS (v4), Recharts.
- **Backend**: Node.js, Express, MongoDB.

## Setup & Run

### Backend
1. Navigate to `backend` folder.
2. Install dependencies: `npm install`.
3. Start server: `npm run dev` (or `node server.js`).
   - Runs on `http://localhost:5000`.

### Frontend
1. Navigate to `frontend` folder.
2. Install dependencies: `npm install`.
3. Start dev server: `npm run dev`.
   - Runs on `http://localhost:5173` (or next available port).

## Credentials
- Register a new account to start using the app.
